# Chess98

Chess98是一个C++实现的中国象棋AI项目，适用于Windows和Linux。
此项目是一个学习项目，主要借鉴象眼引擎，历经两年迭代，棋力提升到了不错的水平。
引擎自带界面程序，且支持UCCI协议，可以通过内置界面或自己喜欢的界面程序下棋。
访问本项目的[发布页](https://github.com/StarlightChessOrg/Chess98/releases/)下载最新版本。

Chess98 is a C++ Chinese Chess AI Project inspired by ElephantEye with Windows and Linux support.
This project was developed with a learning usage and we have been continuously making efforts for about two years.
The strength of the engine finally reaches a good level.
Meanwhile, the built-in method of playing xiangqi as well as a UCCI protocol is realized.
It's recommended to use the integrated UI or the UI app you like to play xiangqi.
Visit our [Release Page](https://github.com/StarlightChessOrg/Chess98/releases/) to download the latest version.

## ⚔️ Usage | 使用

v1.1.0以后的发布将始终提供一个压缩包，带有Chess98的Windows、Linux程序、开局库文件以及ui。

The releases after v1.1.0 will always provide a zip file which contains Chess98.exe, Chess98.out, BOOK.DAT and ui.

内置的UI需要Node环境，如果没有Node环境请到其[官网](https://nodejs.org/)下载。

Integrated UI requires a node envirnment. If you don't have one, you can download it on its [site](https://nodejs.org/).

也可以使用一个界面，这里推荐一个优秀的界面[TChess](https://github.com/sojourners/public-Xiangqi/releases)。

You can choose to use a UI app. Here is a good one called [TChess](https://github.com/sojourners/public-Xiangqi/releases).
