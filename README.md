# Chess98 | 车九平八

Chess98 is a Chinese Chess AI Project written by C++ which inspired by ElephantEye, another excellent 
Xiangqi Engine, with Windows and Linux support. This project was set up with a learning usage and it 
has been continuously developed for about 2 years along with its strength progressing to a not bad level.
And the built-in method of playing xiangqi as well as popular UCCI protocol is realized so that it is OK 
to use the integrated UI or the UI app you like to play xiangqi.

Visit our [GitHub Release](https://github.com/StarlightChessOrg/Chess98/releases) to download the latest version.


一个C++实现的中国象棋AI项目，适用于Windows和Linux。此项目是一个学习项目，借鉴象眼引擎，历经两年迭代，棋力一步步提升到不错水平。
引擎支持UCCI协议和自建的一种下棋方式，可以通过内置界面或自己喜欢的界面程序下棋。

访问本项目的[GitHub Release](https://github.com/StarlightChessOrg/Chess98/releases)下载最新版本。
